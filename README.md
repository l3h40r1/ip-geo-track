# ip-geo-track
  This tool is used to track geo-location of given ip-address or url

# Requirements:
  python3
  
  requests==2.23.0 0r above


# Usage:
  ip-track.py [-h] -i IP

# optional arguments:
  -h, --help      show this help message and exit

# required argument:
  -i IP, --ip IP  Type ip-address or url

# Example: 
  ./ip-track.py -i demo.testfire.net
